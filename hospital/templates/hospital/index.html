Hospital U.S.S.D. Appointment System

The project uses Africaâ€™s Talking API to create a Hospital Appointment System. A patient is able to use a U.S.S.D code to book his/her Appointment. If the process is successful, a booking code is generated for that particular Appointment. The booking code is later used to verify the patient's Appointment at the Hospital.

Steps to take

Step 1

Adding a Patient

  • Click on Patients tab
  • Click on "Add" to add new patient
  • Enter Patient first name
  • Enter Patient last name
  • Enter Patient phone number with their country codes(+254731...)
  • Enter Patient email address
  • Enter Patient username
Step 2

Adding a Doctor

  • Navigate to Doctors tab
  • Click on "Add" to add new doctor
  • Enter doctor first name
  • Enter doctor last name
  • Select doctor phone number with their coutry codes(+254723...)
  • Enter doctor email address
  • Enter doctor username
Step 3

Adding a Service

  • Navigate to Services tab
  • Click on "Add" to add new service
  • Enter Service name
  • Enter Service description
Step 4

Adding a Doctor Service

  • Click on doctor services menu
  • Click on "Add" to link a doctor to a service
  • Select doctor
  • Select service
Step 5

Adding a Doctor Time Slot

  • Click on doctor time slots menu
  • Click on "Add" to provide appointment date for a doctor's service
  • Select doctor offering a particular service
  • Enter the start date
  • Enter the end date

Booking Appointments

Steps for a patient to make a doctor's appointment:

  • Navigate to Africa's Talking Simulator URL: https://simulator.africastalking.com:1517/
  • Enter the correct patient's phone number
  • Click on "USSD"
  • Enter the USSD code you had created e.g. *384*58965#
  • Select a service you linked to a doctor
  • Select the doctor offering the service, since you can multiple doctors offering the same service
  • Select the date you want to book the appointment
  • You will be notified of a successful appointment booking

Viewing appointments made:

  • Click on appointments menu
  • Click on "View" to view appointments made
